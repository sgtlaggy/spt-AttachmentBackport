# AttachmentBackport

> A mod that backports the recent Attachment changes to SPT.

**Author:** RuKira

**Version:** 1.0.0

**SPT Version:** 4.0.13

**License:** MIT

---

## What This Mod Does

Starting from the update `1.0.2.5.43415` on March 5th, Attachments have been edited in-game.

> This ONLY includes **Ergo**, **CheckTime**, **Load & Unload Time**, and **Recoil** stats.

This includes Attachment updates from versions:

[`1.0.2.5.43415` on March 5th (6am EST)](https://changes.tarkov-changes.com/view/989) - [`1.0.2.5.43415` on March 5th (10am EST)](https://changes.tarkov-changes.com/view/991)

[`1.0.4.5.44573` on April 20th (10am EST)](https://changes.tarkov-changes.com/view/1024) - [`1.0.4.5.44573` on April 20th (12pm EST)](https://changes.tarkov-changes.com/view/1025)

THERE SHOULD BE A TOTAL OF **16** MISSING ITEMS | **5** if using Content Backport

## Requirements

- [SPT](https://www.sp-tarkov.com/) **4.0.13** or compatible
- **OPTIONAL** [WTT-CommonLib](https://forge.sp-tarkov.com/mod/2310/wtt-commonlib) **2.0.11** or compatible
- **OPTIONAL** [WTT-ContentBackport](https://forge.sp-tarkov.com/mod/2512/wtt-content-backport) **1.0.4** or compatible
